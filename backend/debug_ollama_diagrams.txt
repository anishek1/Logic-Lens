Note that I've used the Mermaid syntax to create three different diagrams:

* `class_diagram`: shows the class relationships between Streamlit, LangChain, and Groq LLM.
* `flowchart`: illustrates the main application flow from user input to chatbot response.
* `architecture`: depicts the high-level architecture of the system using a microservices pattern.
