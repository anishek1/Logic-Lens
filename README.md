# LogicLens

AI-Powered Code Intelligence Platform - Analyze, understand, and document any codebase.

## Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Features

- 🔍 **Code Analysis** - AI-powered codebase understanding
- 💬 **Chat with Code** - Ask questions about any repository
- 📊 **Diagrams** - Auto-generated Mermaid diagrams
- 📄 **Documentation** - Generate README and docs

## Tech Stack

- **Frontend:** React + Vite + TailwindCSS
- **Backend:** FastAPI + LangChain
- **LLM:** Google Gemini
- **Diagrams:** Mermaid.js
